# ACCOUNTING 002: LEDGER MODEL

## Purpose

Defines the ledger model.

## Ledger Types

- Organization Ledger
- Portfolio Ledger
- Property Ledger
- Unit Ledger
- Lease Ledger
- Vendor Ledger
- Security Deposit Ledger

## Ledger Rule

Every financial transaction should belong to at least one ledger context.

## Examples

Rent payment:

- Organization
- Property
- Unit
- Lease
- Tenant

Vendor payment:

- Organization
- Property
- Work Order
- Vendor
- Expense Category
