# ACCOUNTING 003: SECURITY DEPOSIT ACCOUNTING

## Purpose

Defines security deposit accounting requirements.

## Required Tracking

- Deposit amount
- Date received
- Tenant
- Lease
- Unit
- Property
- Holding account
- Deductions
- Interest where required
- Return date
- Return method
- Supporting documents

## Rule

Security deposits should never be treated as ordinary income.
