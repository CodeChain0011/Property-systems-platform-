# AI 001: ASSISTANT ARCHITECTURE

## Purpose

Defines the AI assistant layer for the Property Operating System.

## Goals

The AI should help users:

- Understand
- Decide
- Act
- Monitor
- Optimize

## Assistant Types

### Tenant Assistant

Supports tenants.

Examples:

- Lease questions
- Rent questions
- Maintenance status
- Move-in guidance

### Manager Assistant

Supports property managers.

Examples:

- Workflow recommendations
- Compliance reminders
- Vendor suggestions
- Risk alerts

### Owner Assistant

Supports owners and investors.

Examples:

- Portfolio performance
- Occupancy analysis
- Cash flow analysis
- Risk monitoring

## AI Rule

The AI recommends.

Humans approve.
