# PERMISSION 001: ACCESS MODEL

## Purpose

Defines access control strategy.

## Access Dimensions

- Organization
- Portfolio
- Property
- Unit
- Role
- Workflow
- Document sensitivity
- Financial visibility

## Rule

Access must be explicit and role-based.

## Examples

Tenant can only access their own lease, payments, documents, and maintenance requests.

Vendor can only access assigned work orders.

Owner can access portfolio records.

Admin can manage users and permissions.
