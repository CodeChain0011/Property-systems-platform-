# WORKFLOW 007: INSPECTION LIFECYCLE

## Purpose

Defines how inspections are created, completed, documented, and converted into action.

## Trigger

Inspection is scheduled manually, by lease event, by compliance rule, or by maintenance schedule.

## Main Actors

- Inspector
- Property Manager
- Tenant
- Vendor
- System

## Workflow Steps

1. Inspection type is selected.
2. Inspection checklist is generated.
3. Inspection is scheduled.
4. Tenant or occupant is notified if required.
5. Inspector completes checklist.
6. Photos and notes are uploaded.
7. Findings are recorded.
8. Required corrections are identified.
9. Work Orders are generated if needed.
10. Compliance records are updated.
11. Inspection is closed.

## Records Created

- Inspection
- Document
- Work Order
- Communication
- Event
- Audit Log
