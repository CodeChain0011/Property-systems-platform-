# TECH ARCH 003: NEO4J GRAPH STRUCTURE

## Purpose

Defines Neo4j's role in the platform.

## Neo4j Responsibilities

Neo4j stores relationship intelligence.

## Graph Use Cases

- Property relationship mapping
- Tenant history
- Lease relationships
- Vendor performance
- Asset dependency mapping
- Compliance context mapping
- Maintenance pattern analysis
- Portfolio intelligence
- AI context graph

## Neo4j Rule

Neo4j should not replace PostgreSQL as the official record.

Neo4j should enrich the platform by making complex relationships easier to query and understand.
