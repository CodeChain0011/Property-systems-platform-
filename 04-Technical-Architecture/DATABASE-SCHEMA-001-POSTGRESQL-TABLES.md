# DATABASE SCHEMA 001: POSTGRESQL TABLES

## Purpose

Defines the first planned PostgreSQL table groups.

## Core Table Groups

### Account Tables

- organizations
- users
- people
- roles
- permissions

### Property Tables

- portfolios
- properties
- buildings
- units

### Leasing Tables

- applications
- leases
- lease_parties
- lease_terms
- lease_documents

### Maintenance Tables

- assets
- work_orders
- inspections
- maintenance_schedules

### Financial Tables

- ledgers
- financial_transactions
- invoices
- payments
- security_deposits

### Communication Tables

- communications
- notices
- message_templates

### System Tables

- events
- audit_logs
- files
- notifications

## Rule

PostgreSQL is the official source of truth.
