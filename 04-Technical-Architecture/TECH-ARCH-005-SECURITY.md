# TECH ARCH 005: SECURITY FOUNDATION

## Purpose

Defines the first security principles for the platform.

## Security Areas

- Authentication
- Authorization
- Role-based access
- Data encryption
- Audit logging
- Document protection
- Tenant privacy
- Financial data protection
- Admin controls
- Session management

## Security Rule

Every sensitive action must answer:

- Who performed it?
- What changed?
- When did it happen?
- What record was affected?
- Was the user authorized?
