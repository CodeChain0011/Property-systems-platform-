# TECH ARCH 001: SYSTEM OVERVIEW

## Purpose

Defines the high-level technical architecture for the Property Operating System.

## Core Layers

1. Web Application
2. Mobile Application
3. API Backend
4. PostgreSQL Database
5. Neo4j Graph Database
6. Object Storage
7. Authentication Layer
8. Permission Layer
9. Workflow Engine
10. Notification Service
11. AI Assistance Layer
12. Reporting Layer
13. Audit Logging Layer

## Design Rule

Each technical layer must support small landlords and enterprise property operators without requiring a separate architecture.
