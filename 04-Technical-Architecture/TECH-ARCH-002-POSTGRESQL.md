# TECH ARCH 002: POSTGRESQL STRUCTURE

## Purpose

Defines PostgreSQL's role in the platform.

## PostgreSQL Responsibilities

PostgreSQL stores the official system of record.

## Primary Data Categories

- Organizations
- Users
- Roles
- Permissions
- Properties
- Buildings
- Units
- Leases
- Applications
- Work Orders
- Inspections
- Documents Metadata
- Financial Transactions
- Communications
- Events
- Audit Logs

## PostgreSQL Rule

If data must be legally accurate, transactionally reliable, reportable, or auditable, it belongs in PostgreSQL.
