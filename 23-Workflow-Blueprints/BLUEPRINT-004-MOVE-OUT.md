# BLUEPRINT 004: MOVE-OUT

## Purpose

Map the move-out process.

## Flow

1. Move-out notice is received or lease ends.
2. Move-out date is confirmed.
3. Move-out instructions are sent.
4. Final inspection is scheduled.
5. Tenant returns access items.
6. Inspection is completed.
7. Damage is documented.
8. Deposit deductions are calculated.
9. Deposit return package is prepared.
10. Turnover work orders are created.
11. Unit status becomes turnover or available.

## Records Created

- Notice
- Inspection
- Document
- Work Order
- Financial Transaction
- Communication
- Event
- Audit Log

## Automation Opportunities

- Deposit deadline tracking
- Damage comparison
- Turnover checklist
- Cleaning/vendor scheduling
