# BLUEPRINT 001: APPLICANT TO TENANT

## Purpose

Map the complete journey from rental interest to active tenancy.

## Flow

1. Prospect views listing.
2. Prospect submits inquiry.
3. System creates lead record.
4. Prospect submits application.
5. System creates applicant record.
6. Screening begins.
7. Documents are collected.
8. Manager reviews application.
9. Decision is recorded.
10. Approved applicant receives lease.
11. Lease is signed.
12. Deposit is collected.
13. Move-in is scheduled.
14. Role changes from Applicant to Tenant.
15. Unit status updates.

## Records Created

- Lead
- Person
- Role
- Application
- Document
- Screening Record
- Lease
- Financial Transaction
- Communication
- Event
- Audit Log
