# ACCEPTANCE GATES

## Gate 1: Architecture Ready

Data model, workflows, requirements, and screens exist.

## Gate 2: Development Ready

APIs, database specs, and tests are defined.

## Gate 3: MVP Ready

Core landlord workflows work.

## Gate 4: Pilot Ready

A real user can operate a real property.

## Gate 5: Production Ready

Security, backup, monitoring, and support are in place.
