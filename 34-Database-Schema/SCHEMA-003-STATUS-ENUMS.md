# SCHEMA 003: STATUS ENUMS

## Purpose

Defines first system statuses.

## Unit Status

- vacant
- occupied
- turnover
- maintenance
- unavailable

## Lease Status

- draft
- pending_signature
- active
- expired
- terminated

## Work Order Status

- new
- assigned
- in_progress
- waiting
- completed
- closed

## Rule

Statuses must support reporting and workflow automation.
