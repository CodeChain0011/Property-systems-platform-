# SEARCH 004: SEMANTIC SEARCH

## Purpose

Allow concept-based searching.

## Examples

Search:

"water leaks"

May return:

- Plumbing Work Orders
- Pipe Assets
- Inspection Findings
- Vendor Reports

Search:

"late rent"

May return:

- Delinquent Leases
- Notices
- Communications
- Financial Records
