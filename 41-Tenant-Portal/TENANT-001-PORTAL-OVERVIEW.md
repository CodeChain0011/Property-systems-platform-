# TENANT PORTAL OVERVIEW

## Purpose

The Tenant Portal gives residents a self-service interface for lease records, maintenance, payments, documents, notices, and communication.

## Goals

- Reduce manual calls and texts
- Improve tenant experience
- Create better records
- Centralize communications
- Support faster maintenance response

## Core Areas

- Dashboard
- Lease
- Payments
- Maintenance
- Documents
- Notices
- Messages
- Profile
