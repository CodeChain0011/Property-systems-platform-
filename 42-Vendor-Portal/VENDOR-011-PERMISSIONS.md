# VENDOR PORTAL PERMISSIONS

## Vendor Can

- View assigned work orders
- Update assigned work order status
- Upload completion evidence
- Submit invoices
- Send work-order messages
- Update limited profile information

## Vendor Cannot

- View unrelated properties
- View tenant financials
- View owner financials
- View other vendors
- Change approved invoice amounts
- Access internal notes

## Rule

Vendor access must be narrow and auditable.
