# VENDOR WORK ORDERS

## Purpose

Allow vendors to view and update assigned work.

## Visible Information

- Work Order Number
- Property or Unit Access Details
- Issue Category
- Priority
- Description
- Photos
- Appointment Window
- Status
- Required Completion Evidence

## Vendor Actions

- Accept Work
- Decline Work
- Request More Information
- Schedule Visit
- Update Status
- Upload Photos
- Mark Complete

## Rule

Vendors can only access work orders assigned to them.
