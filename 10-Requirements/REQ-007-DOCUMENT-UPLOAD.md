# REQ 007: DOCUMENT UPLOAD

## Requirement ID

REQ-DOC-001

## User Role

Owner or Property Manager

## Goal

Upload and attach a document to a record.

## Main Flow

1. User opens Document Vault.
2. User selects Upload Document.
3. User chooses file.
4. User selects document type.
5. User links document to property, unit, tenant, lease, or work order.
6. System stores file.
7. System creates Document metadata.
8. System creates Event.
9. System creates Audit Log.

## Data Created

- Document
- Event
- Audit Log

## Acceptance Criteria

- File uploads successfully.
- Document metadata is created.
- Document is linked to selected record.
- Audit log is created.

## Priority

High
