# REQ 003: UNIT SETUP

## Requirement ID

REQ-UNIT-001

## User Role

Owner or Property Manager

## Goal

Create a unit under a property.

## Main Flow

1. User opens Property Detail.
2. User selects Add Unit.
3. User enters unit details.
4. System validates required fields.
5. System creates Unit record.
6. System links Unit to Property.
7. System creates Event.
8. System creates Audit Log.

## Data Created

- Unit
- Event
- Audit Log

## Acceptance Criteria

- User can create unit.
- Unit appears under property.
- Unit has occupancy status.
- Audit log is created.

## Priority

High
