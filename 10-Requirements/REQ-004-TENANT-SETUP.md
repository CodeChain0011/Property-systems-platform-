# REQ 004: TENANT SETUP

## Requirement ID

REQ-TENANT-001

## User Role

Owner or Property Manager

## Goal

Create a tenant/person record.

## Main Flow

1. User opens People or Tenants.
2. User selects Add Tenant.
3. User enters contact details.
4. System creates Person record.
5. System assigns Tenant role.
6. System creates Event.
7. System creates Audit Log.

## Data Created

- Person
- Role
- Event
- Audit Log

## Acceptance Criteria

- User can create tenant.
- Tenant appears in tenant list.
- Tenant has Person record.
- Tenant role is assigned.

## Priority

High
