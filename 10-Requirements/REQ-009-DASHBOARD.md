# REQ 009: BASIC DASHBOARD

## Requirement ID

REQ-DASH-001

## User Role

Owner or Property Manager

## Goal

View basic operational status.

## Dashboard Widgets

- Total properties
- Total units
- Occupied units
- Vacant units
- Open work orders
- Rent received this month
- Recent activity

## Acceptance Criteria

- Dashboard loads successfully.
- Dashboard shows accurate counts.
- User can click from widget to detail page.
- Data respects permissions.

## Priority

High
