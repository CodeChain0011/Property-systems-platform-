# REQ 008: BASIC FINANCIAL LOG

## Requirement ID

REQ-FIN-001

## User Role

Owner or Property Manager

## Goal

Record basic income or expense.

## Main Flow

1. User opens Financial Log.
2. User selects Add Transaction.
3. User enters amount, date, category, and related property/unit.
4. System creates Financial Transaction.
5. System links transaction to related records.
6. System creates Event.
7. System creates Audit Log.

## Data Created

- Financial Transaction
- Event
- Audit Log

## Acceptance Criteria

- User can record income.
- User can record expense.
- Transaction links to property or unit.
- Audit log is created.

## Priority

High
