# KPI LIBRARY

## Purpose

Define key performance indicators.

## Core KPIs

- Occupancy Rate
- Vacancy Rate
- Rent Collection Rate
- Delinquency Rate
- Open Work Orders
- Average Work Order Completion Time
- Maintenance Cost Per Unit
- Net Cash Flow
- Compliance Risk Count
- Lease Expiration Count

## Rule

KPIs must trace back to source records.
