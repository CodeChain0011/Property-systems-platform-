## HOW TO USE THIS FORM

For each action row, mark each role column with:
- **Y** = yes, this role can always perform this action
- **N** = no, this role cannot perform this action
- **O** = own records only (e.g. tenant can read their own lease, not others)
- **A** = with approval (another role must approve first)

### Role Column Key
```
OWN = Owner / Administrator
PM  = Property Manager
LA  = Leasing Agent
MC  = Maintenance Coordinator
TEN = Tenant
VEN = Vendor
INS = Inspector
ACC = Accountant
ATT = Attorney
```
