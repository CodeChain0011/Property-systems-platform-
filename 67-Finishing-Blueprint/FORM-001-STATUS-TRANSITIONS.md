## HOW TO USE THIS FORM

For each transition table below:
- Write **Y** in the cell if the transition is ALLOWED
- Write **N** if the transition is BLOCKED
- Write the role abbreviations who can trigger it (see key at top)
- Use the Notes column for side effects (e.g. "updates unit occupancy")

### Role Key (use abbreviations in cells)
```
OWN = Owner / Administrator
PM  = Property Manager
LA  = Leasing Agent
MC  = Maintenance Coordinator
TEN = Tenant
VEN = Vendor
ANY = Any authenticated user
```
