## HOW TO USE THIS FORM

For each field row:
- **Req?** — circle Y (required) or N (optional)
- **Format / Allowed Values** — write the constraint (e.g. "email format", "1–28", "positive number", "max 500 chars")
- **Business Rule** — write any logic beyond format (e.g. "must be after start date", "cannot change once active")

Leave a cell blank if you have no constraint for that field.
