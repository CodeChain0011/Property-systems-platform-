# MVP 002: FIRST WORKFLOWS

## Purpose

Defines the first workflows that should be built.

## First Workflows

### 1. Create Property

User creates a property record.

### 2. Create Unit

User creates one or more units under a property.

### 3. Add Tenant

User creates a tenant/person record.

### 4. Create Lease Record

User links tenant to unit through lease record.

### 5. Upload Document

User uploads lease or related file.

### 6. Create Maintenance Request

User creates a work order.

### 7. Close Maintenance Request

User marks work order completed and adds cost.

### 8. Record Rent Payment

User records rent payment.

## MVP Workflow Rule

Every MVP workflow must update the canonical data model.
