# MVP 004: SUCCESS CRITERIA

## Purpose

Defines how the first MVP will be judged.

## Success Criteria

The MVP is successful if a user can:

1. Create an organization
2. Add a property
3. Add units
4. Add tenants
5. Create lease records
6. Upload lease documents
7. Create work orders
8. Track work order status
9. Record rent payments
10. View basic dashboard status

## Non-Technical Success

The user should understand:

- What properties exist
- Which units are occupied
- Who the tenants are
- Which leases are active
- What maintenance is open
- What rent has been received

## MVP Rule

If the system cannot clearly answer these basic questions, it is not ready for advanced features.
