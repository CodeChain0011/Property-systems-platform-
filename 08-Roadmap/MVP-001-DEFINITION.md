# MVP 001: FIRST BUILDABLE VERSION

## Purpose

Defines the first minimum viable product for the Property Operating System.

## MVP Goal

Build the smallest useful version of the platform that helps a landlord manage real property operations.

## Target First User

Small landlord or owner-operator managing:

- 1 to 10 properties
- Residential units
- Basic tenants
- Basic leases
- Basic maintenance
- Basic documents
- Basic financial tracking

## MVP Core Features

1. Account setup
2. Organization setup
3. Property setup
4. Unit setup
5. Person setup
6. Tenant record
7. Lease record
8. Document upload
9. Maintenance request
10. Work order tracking
11. Basic rent tracking
12. Basic dashboard

## MVP Exclusions

Do not build these first:

- Enterprise multi-state compliance automation
- Full accounting system
- AI recommendation engine
- Advanced analytics
- Vendor marketplace
- Automated eviction workflow
- Complex rent control engine

## MVP Rule

The MVP should prove the core operating model before expanding into enterprise complexity.
