# SECURITY 005: DATA RETENTION

## Purpose

Defines data retention principles.

## Retention Categories

- Active operational records
- Historical lease records
- Financial records
- Applicant records
- Inspection records
- Work order records
- Communications
- Legal notices
- Audit logs
- Deleted user records

## Retention Rule

Records should not be deleted casually. The system must support retention schedules, legal holds, and controlled deletion.
