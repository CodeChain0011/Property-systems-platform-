# SECURITY 001: SECURITY OVERVIEW

## Purpose

Defines the security foundation for the Property Operating System.

## Security Areas

- Authentication
- Authorization
- Role-based access control
- Tenant data protection
- Financial data protection
- Document protection
- Audit logging
- Session management
- Admin controls
- Data retention
- Legal hold
- Backup and recovery

## Security Rule

Every sensitive action must be traceable, permission-controlled, and auditable.
