# SECURITY 003: SENSITIVE DATA

## Purpose

Defines sensitive data categories.

## Sensitive Data Types

- Tenant identity records
- Applicant screening data
- Lease documents
- Financial transactions
- Bank/payment records
- Tax records
- Vendor tax forms
- Legal notices
- Private communications
- Inspection media
- Security camera records

## Protection Rule

Sensitive data must be access-controlled, encrypted where appropriate, logged when accessed, and retained according to policy.
