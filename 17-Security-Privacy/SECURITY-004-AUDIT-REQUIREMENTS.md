# SECURITY 004: AUDIT REQUIREMENTS

## Purpose

Defines audit logging requirements.

## Must Audit

- Login events
- Failed login attempts
- Permission changes
- Role changes
- Lease edits
- Financial edits
- Document uploads
- Document deletions
- Notice generation
- Payment approvals
- Security deposit deductions
- User invitations
- Data exports

## Audit Rule

Audit logs should be difficult to alter and should preserve who did what, when, and to which record.
