# DOCUMENT SCALE

## Purpose

Support large document volume.

## Document Types

- Leases
- Addenda
- Invoices
- Receipts
- Notices
- Photos
- Videos
- Inspection Reports
- Compliance Records

## Requirements

- Object storage
- Metadata indexing
- Retention policies
- Search
- Permissions
- Versioning

## Rule

Documents should not live directly inside the relational database.
