# GRAPH 002: RELATIONSHIP TYPES

## Purpose

Defines the first Neo4j relationship types.

## Relationship Types

- OWNS
- MANAGES
- CONTAINS
- BELONGS_TO
- HAS_ROLE
- HAS_PERMISSION
- GOVERNS
- SIGNED_BY
- OCCUPIES
- REQUESTED_BY
- ASSIGNED_TO
- DOCUMENTS
- GENERATED
- RELATED_TO
- PAID_BY
- PAID_TO
- TRIGGERED
- CREATED_BY
- UPDATED_BY
- AUDITED_BY

## Rule

Relationship names should describe business meaning, not database mechanics.
