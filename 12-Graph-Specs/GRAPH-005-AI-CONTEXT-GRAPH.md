# GRAPH 005: AI CONTEXT GRAPH

## Purpose

Defines how Neo4j supports AI context.

## AI Context Sources

- Entity relationships
- Events
- Communications
- Documents
- Work orders
- Financial records
- Lease records
- Compliance records

## AI Use Cases

- Explain property status
- Summarize tenant history
- Identify maintenance risk
- Find supporting documents
- Recommend next action
- Surface operational risk

## Rule

The AI assistant should use the graph to understand context before generating responses.
