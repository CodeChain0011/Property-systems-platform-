# GRAPH 004: QUERY USE CASES

## Purpose

Defines why Neo4j is useful in this platform.

## Example Questions

- Which units have the most maintenance issues?
- Which assets create the most expense?
- Which vendors are connected to repeated failures?
- Which tenants have active leases and open maintenance requests?
- Which properties have compliance risk?
- Which documents support a specific dispute?
- Which units are connected to late rent and high maintenance cost?

## Graph Rule

Use Neo4j when the question depends on relationships across multiple entities.
