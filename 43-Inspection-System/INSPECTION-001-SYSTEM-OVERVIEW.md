# INSPECTION SYSTEM OVERVIEW

## Purpose

The Inspection System captures property condition, compliance status, asset health, move-in/move-out evidence, and maintenance findings.

## Goals

- Protect landlord records
- Protect tenant records
- Support security deposit decisions
- Trigger maintenance
- Support compliance
- Create photo/video evidence
- Track asset condition over time

## Core Inspection Types

- Move-In Inspection
- Move-Out Inspection
- Annual Inspection
- Safety Inspection
- Maintenance Inspection
- Insurance Inspection
- Vendor Completion Inspection
