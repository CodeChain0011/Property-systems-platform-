# TEST 003: PERMISSION TESTING

## Purpose

Defines permission testing requirements.

## Required Permission Tests

- Tenant cannot view owner financials.
- Vendor cannot view unrelated properties.
- Manager cannot access organizations they are not assigned to.
- Owner can view properties they own.
- Admin can manage users.
- Sensitive documents respect access level.
- Financial actions require proper permission.

## Rule

Every permission must be tested from both allowed and denied perspectives.
