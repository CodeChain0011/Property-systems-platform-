# TEST 002: MVP TEST CASES

## Purpose

Defines initial MVP test cases.

## Required MVP Tests

### Property

- User can create property.
- User can edit property.
- User can view property list.

### Unit

- User can create unit.
- Unit links to property.
- Unit occupancy status updates.

### Tenant

- User can create tenant.
- Tenant receives correct role.
- Tenant links to lease.

### Lease

- User can create lease.
- Lease links tenant and unit.
- Lease updates unit occupancy status.

### Work Order

- User can create work order.
- Work order links to property and unit.
- Work order status can change.

### Document

- User can upload document.
- Document metadata is saved.
- Document links to selected record.

### Financial Log

- User can create transaction.
- Transaction links to property or unit.
