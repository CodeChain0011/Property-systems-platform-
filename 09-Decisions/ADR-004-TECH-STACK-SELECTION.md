# ADR-004: TECH STACK SELECTION

## Decision

The Property Operating System will use a modern hybrid web architecture.

## Proposed Stack

### Frontend

- React
- Next.js
- TypeScript

### Backend

- NestJS or FastAPI

### Primary Database

- PostgreSQL

### Graph Database

- Neo4j

### File Storage

- S3-compatible object storage

### Authentication

- Role-based authentication and authorization

### Hosting

- Cloudflare
- VPS or cloud backend
- Managed database services when appropriate

## Reason

This stack supports:

- Web application
- Mobile expansion
- API-first architecture
- Strong database structure
- Graph intelligence
- AI integration
- Long-term scalability

## Rule

Final stack choices must support both small landlords and enterprise operators.
