# COMPLIANCE ENGINE 001: OVERVIEW

## Purpose

Defines the foundation for a jurisdiction-aware compliance engine.

The platform must eventually understand that landlord rules change by:

- Country
- State
- County
- City
- Property type
- Unit type
- Lease type
- Rent control status

## Core Goal

The system should help determine which legal, financial, notice, inspection, and document rules apply to a specific property.

## Rule

Compliance is location-aware.

No legal notice, rent rule, deposit rule, or deadline should be generated without jurisdiction context.
