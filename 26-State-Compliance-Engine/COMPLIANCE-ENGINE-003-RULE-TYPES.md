# COMPLIANCE ENGINE 003: RULE TYPES

## Purpose

Defines the types of rules the compliance engine must eventually support.

## Rule Categories

- Security Deposit Rules
- Rent Increase Rules
- Notice Period Rules
- Eviction Process Rules
- Entry Notice Rules
- Habitability Rules
- Fair Housing Rules
- Application Screening Rules
- Late Fee Rules
- Lease Disclosure Rules
- Move-Out Rules
- Inspection Rules
- Record Retention Rules

## Rule Structure

Each rule should define:

- Rule name
- Jurisdiction
- Effective date
- Applies to
- Trigger
- Requirement
- Deadline
- Required document
- Required notice
- Source reference
- Risk level
