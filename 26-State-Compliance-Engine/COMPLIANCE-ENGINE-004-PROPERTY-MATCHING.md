# COMPLIANCE ENGINE 004: PROPERTY MATCHING

## Purpose

Defines how a property should be matched to applicable rules.

## Inputs

- Property address
- City
- County
- State
- Zip code
- Parcel number
- Property type
- Unit count
- Year built
- Rent control status
- Ownership type

## Matching Process

1. Identify state.
2. Identify county.
3. Identify city.
4. Identify special housing zones.
5. Identify property type.
6. Identify applicable rule sets.
7. Attach compliance context to property.

## Rule

Compliance context must be attached to the property record.
