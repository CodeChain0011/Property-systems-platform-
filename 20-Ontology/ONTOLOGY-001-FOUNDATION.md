# PROPERTY OPERATING SYSTEM ONTOLOGY

## Purpose

Defines the conceptual language of the platform.

## Ontology Rule

Every object, workflow, report, AI decision, and database record must trace back to a defined concept.

## Core Concept Categories

- Organization
- Place
- Person
- Agreement
- Asset
- Event
- Task
- Communication
- Financial Object
- Compliance Object
- Document
- Relationship
