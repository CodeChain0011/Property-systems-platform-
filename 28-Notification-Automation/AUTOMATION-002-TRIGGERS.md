# AUTOMATION 002: TRIGGERS

## Purpose

Defines automation trigger types.

## Trigger Categories

- Date Trigger
- Event Trigger
- Status Change Trigger
- Financial Trigger
- Compliance Trigger
- Document Trigger
- Inspection Trigger
- Maintenance Trigger
- Lease Trigger
- Tenant Trigger

## Examples

Lease expires in 90 days.

Rent payment becomes overdue.

Insurance expires in 30 days.

Work order exceeds deadline.

Inspection finds violation.
