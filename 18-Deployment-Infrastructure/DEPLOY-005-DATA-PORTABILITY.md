# DEPLOY 005: DATA PORTABILITY

## Purpose

Defines data portability principles.

## Exportable Data

- Organizations
- Properties
- Units
- People
- Leases
- Work Orders
- Documents metadata
- Financial transactions
- Communications
- Audit logs

## Export Formats

- CSV
- JSON
- PDF where appropriate

## Portability Rule

The customer should be able to export their business records in standard formats.
