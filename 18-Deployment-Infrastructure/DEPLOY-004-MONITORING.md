# DEPLOY 004: MONITORING

## Purpose

Defines monitoring requirements.

## Monitor

- API uptime
- Web app uptime
- Database health
- Graph database health
- Storage health
- Error rates
- Failed jobs
- Failed notifications
- Security events
- Backup success

## Monitoring Rule

If a failure affects users, records, payments, compliance, or security, it must be visible quickly.
