# DOMAIN 002: FAIR HOUSING

## Purpose

Defines fair housing requirements and protections.

## Core Principles

- Consistent application processing
- Consistent screening standards
- Consistent approval criteria
- Consistent denial criteria
- Protected class awareness
- Documentation requirements

## Platform Requirements

Every applicant decision must be:

- Traceable
- Explainable
- Auditable
- Documented
