# DOMAIN 003: SECURITY DEPOSITS

## Purpose

Defines security deposit handling requirements.

## Lifecycle

1. Deposit collected
2. Deposit stored
3. Deposit tracked
4. Move-out inspection completed
5. Deductions evaluated
6. Reconciliation prepared
7. Deposit returned

## Required Records

- Deposit amount
- Deposit date
- Holding account
- Deduction records
- Inspection evidence
- Return date
