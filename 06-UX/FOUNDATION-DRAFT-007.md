# PROPERTY OPERATING SYSTEM

## FOUNDATION DRAFT 007

### User Experience Foundation

This document defines the UX direction for the platform.

## Primary User Types

1. Individual landlord
2. Property owner
3. Property manager
4. Tenant
5. Applicant
6. Vendor
7. Maintenance worker
8. Inspector
9. Accountant
10. Attorney
11. Enterprise administrator

## UX Principle

The system must feel simple to the user even when the backend is complex.

## Core Interfaces

1. Owner dashboard
2. Manager dashboard
3. Tenant portal
4. Applicant portal
5. Vendor portal
6. Maintenance board
7. Financial dashboard
8. Compliance calendar
9. Document vault
10. Portfolio intelligence dashboard

## UX Rule

Every screen must answer:

- What needs attention?
- What is the current status?
- What action can the user take?
- What records will be created?
- What risk or deadline exists?
