# UX 001: USER TYPES

## Purpose

Defines the major users the platform must serve.

## User Types

1. Individual Landlord
2. Small Portfolio Owner
3. Property Manager
4. Enterprise Administrator
5. Tenant
6. Applicant
7. Vendor
8. Maintenance Worker
9. Inspector
10. Accountant
11. Attorney
12. Owner Representative

## UX Rule

Each user type should only see the tools, records, actions, and reports relevant to their role.
