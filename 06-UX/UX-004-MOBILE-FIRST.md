# UX 004: MOBILE-FIRST EXPERIENCE

## Purpose

Defines mobile requirements for landlords, tenants, vendors, and field workers.

## Mobile-Critical Actions

- Submit maintenance request
- Upload photo or video
- Pay rent
- Send message
- View lease
- Complete inspection
- Update work order
- View property status
- Approve urgent action
- Receive deadline alerts

## Mobile Rule

Any task that happens in the field must work well on mobile.
