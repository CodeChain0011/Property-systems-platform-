# UX 002: DASHBOARDS

## Purpose

Defines the major dashboard experiences.

## Core Dashboards

- Owner Dashboard
- Property Manager Dashboard
- Tenant Dashboard
- Applicant Dashboard
- Vendor Dashboard
- Maintenance Dashboard
- Financial Dashboard
- Compliance Dashboard
- Portfolio Intelligence Dashboard
- Admin Dashboard

## Dashboard Rule

Every dashboard must answer:

- What needs attention?
- What changed recently?
- What is overdue?
- What requires approval?
- What risks exist?
- What can be automated?
