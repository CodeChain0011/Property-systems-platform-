# OWNER DASHBOARD

## Purpose

Provide a real-time overview of the entire portfolio.

## Top Metrics

- Total Properties
- Total Units
- Occupied Units
- Vacant Units
- Open Work Orders
- Active Leases
- Rent Collected This Month
- Outstanding Balance

## Dashboard Widgets

### Occupancy Widget

Shows:

- Occupied %
- Vacant %
- Turnover %

### Financial Widget

Shows:

- Revenue
- Expenses
- Net Cash Flow

### Maintenance Widget

Shows:

- Open Work Orders
- High Priority Work Orders
- Overdue Work Orders

### Compliance Widget

Shows:

- Upcoming Deadlines
- Missing Documents
- Compliance Alerts

### AI Widget

Shows:

- Recommended Actions
- Risk Alerts
- Portfolio Health Score
